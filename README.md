# Pizzaria - Atividade 4

## Descrição
Nesta atividade foi criada uma página de contato com validação completa em JavaScript.

---

## Funcionalidades

- Validação de campos obrigatórios
- Validação de e-mail com Regex
- Validação de tamanho mínimo da mensagem
- Mensagens de erro abaixo dos campos
- Bloqueio de envio inválido
- Feedback visual ao usuário

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## Como Executar pelo Terminal

### 1. Abrir o projeto no VSCode ou GitHub Codespaces

### 2. Abrir o terminal

### 3. Executar o comando:

```bash
python3 -m http.server 5500
```

### 4. Abrir no navegador:

```bash
http://localhost:5500
```
